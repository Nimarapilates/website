You are generating marketing material for Re:Connect Reformer Pilates, a premium Reformer Pilates studio in Palma de Mallorca, Spain. Read `brand.yaml` in this directory for the complete brand identity. This file provides generation-specific instructions.

## Before you start

1. Read `brand.yaml` fully — it contains colours, typography, tone of voice, layout rules, and all brand details.
2. Logo files are in the `logos/` directory. Use `reconnect-dark.svg` on light backgrounds, `reconnect-light.svg` on dark backgrounds, and `reconnect-icon.svg` for small-format use.

## Generating images

### Colour palette
- Background: Sand `#F5F0E8` or Cream `#FAF8F4`. Never pure white `#FFF`.
- Headings: Warm Charcoal `#2C2C2C`. Never pure black `#000`.
- Body text: Stone `#8C8577`.
- Accent: Calke Green `#5B7553` for labels, CTAs, highlights, and key UI elements.
- Warm accent: Dune `#C2A77D` for card backgrounds, decorative elements, and warmth.
- Never use bright/saturated colours, neon tones, or any colour outside the defined palette.

### Typography
- Heading font: Cormorant Garamond. If unavailable, use a refined serif (e.g. Playfair Display, Libre Baskerville).
- Body font: DM Sans. If unavailable, use a clean sans-serif (e.g. Inter, Source Sans 3).
- Headings: font-weight 300 (light). Large. Generous line height.
- Labels above headings: ALL CAPS, very small, wide letter-spacing (0.2em), medium weight, Calke Green.
- Body copy: font-weight 400, Stone colour.
- Buttons/CTAs: font-weight 500, Calke Green background, Cream text.

### Layout
- Centre-align text for social media posts. Left-align for website and long-form content.
- Use generous whitespace — the brand is calm, spacious, and breathable.
- Thin green line (2px wide, Calke Green, ~40px long) placed above heading text as a visual marker.
- Include `reconnectpilates.es` or `@reconnect.pilates` on every asset.
- Keep designs clean. One message per asset. Do not overcrowd.
- When layering text on photography, apply a soft warm overlay to maintain readability.

## Writing copy

### Voice
- Calm, warm, assured. Never shouty or salesy.
- Short sentences. Active voice. Concrete details.
- Use "we" and "our" — the studio speaks as a team.
- Sentence case for headings (not Title Case).
- Bilingual: English is the lead language, Spanish translation should always be available.

### Do
- Describe the experience and feeling, not just features.
- Use specific details ("8 Balanced Body Allegro 2 Reformers").
- Use "practice" not "workout" when referring to Pilates.
- Keep paragraphs to 2-3 sentences maximum.
- Reference the Mallorcan context naturally (the light, the neighbourhood, the island).

### Do not
- Use superlatives (best, leading, world-class, unrivalled).
- Use exclamation marks in brand copy.
- Use fitness-culture language (crush, burn, beast mode, gains, shred).
- Use the word "luxury" — let the experience communicate that.
- Use weight-loss or body-shaming language.
- Use buzzwords (solutions, leverage, synergy, passionate, holistic).
- Use pseudo-spiritual language (manifest, vibration, energy healing, chakra).
- Use emojis in brand copy (acceptable sparingly in social media captions).
